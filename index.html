Инструкция

iOS (Happ)

  1. Скачайте Happ из App Store
  2. Откройте приложение → нажмите + → Import from Clipboard
  3. Вставьте скопированный ключ → Сохранить
  4. Включите переключатель

Android (Happ)

  1. Скачайте Happ из Google Play или официального сайта
  2. Откройте → нажмите + → Импорт из буфера обмена
  3. Вставьте ключ → Сохранить
  4. Включите VPN

Windows (Happ)

  1. Скачайте Happ с официального сайта
  2. Откройте приложение → Сервер → Импорт из буфера обмена
  3. Вставьте ключ → Сохранить
  4. Включите подключение

macOS (Happ)

  1. Скачайте Happ с официального сайта
  2. Нажмите на иконку в строке меню → Import Server from Clipboard
  3. Вставьте ключ → Сохранить
  4. Turn On
